Welcome to the hermal version downloader!
Hermal is my little project, like a raspberry pi terminal in a terminal! it requires colorama and time to function, and also, here is what the names mean:

hermal_v1_0001A_C.py
 - 'hermal' is kinda obvious its the name of the thing
 - 'v1' is the version number, in this case version 1
 - '0001A' is the build number, where here its build number 0001 (or just 1), and the A means alpha, so b means beta, too.
 - 'C' just means that its a 'color' version, which means it uses colorama. idk if there will be non-color versions, but ok wow
